> # 🛒 Commerce Project 🛒

> ## 👨‍🏫 프로젝트 소개

#### 이 프로젝트는 객체지향 설계를 기반으로 상품, 카테고리 등의 객체를 정의한다.
#### 각 객체 간 역할과 책임을 나누어 협력하는 구조로 구현한 커머스 플랫폼이다.

> ## 📌 주요기능 

### STEP1. 객체 지향 설계 적용 사품관리 시스템 프로그래밍
 - Product 클래스 생성 및 상품명, 가격, 설명, 재고수량 필드 정의
 - 상품 정보 출력 및 0 입력 시 프로그램 종료

### STEP2. 
 - CommerceSystem 클래스 생성
   - start 함수 구현하여 입력과 반복문 로직 구현 

### STEP3.4. 상품 선택 및 출력 기능 구현
 - 카테고리 선택 후 상품 목록 출력
 - 상품 선택 시 상세 정보 출력
 - 사용자 입력 검증 로직 추가

> ## 🔥 Trouble Shooting
 ### 1. 출력 형식 정리
- `%d` : 정수 출력
- `%s` : 문자열 출력
- `%-15s` : 문자열 왼쪽 정렬 후 15칸 확보
- `%,d` : 천 단위 콤마 적용
- `%n` : 줄바꿈
